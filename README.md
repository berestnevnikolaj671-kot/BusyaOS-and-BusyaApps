# 🚀 BusyaOS and BusyaApps

## 🇬🇧 English

BusyaOS is a lightweight offline-first HTML operating system concept inspired by tile-based mobile interfaces.  
It works locally in a browser and focuses on privacy, customization, multilingual support, and simple local app installation.

BusyaApps is a collection of standalone HTML apps for BusyaOS and regular browsers.

### ✨ Features

- 🟦 Live tile style interface  
- 🌍 Multilingual system (RU / EN / BY)  
- 🎨 Themes and customization  
- 🔒 Privacy-focused local usage  
- 📦 Built-in App Store  
- 📝 Busya 366 Office  
- 📂 File tools  
- 🧮 Calculator  
- ⏱ Timer apps  
- 📷 QR scanner  
- 🌐 Browser  
- ⚙️ First start setup wizard

### 📥 Installation

1. Download repository ZIP or HTML file  
2. Open in browser  
3. Enjoy BusyaOS offline

Recommended browsers: Samsung Internet / Chrome-based browsers.
